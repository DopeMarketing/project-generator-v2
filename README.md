# Project Generator V2

## Overview
Enhanced Project Generator that extends existing functionality with database schema generation, application structure mapping, and collaborative project sharing capabilities. This builds upon the current solo project creation workflow to provide deeper technical specifications and team collaboration features.

## Problem
Current Project Generator provides scope and automation but lacks detailed technical architecture (database design, app structure) and collaboration features needed for team-based project development

## Target User
Product managers, technical founders, and development teams who need comprehensive project specifications and want to collaborate on project planning

## Features
- Postgres schema generation with tables, columns, types, indexes, and RLS policies
- Site map generation with routes, purposes, auth requirements, and components
- Email-based collaborator invitations with read-only access
- Permission system distinguishing project owners from viewers
- Structured output formats for generated schemas and site maps
